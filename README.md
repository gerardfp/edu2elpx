# Edumark to eXeLearning (edu2elpx)

Esta extensión de VS Code permite previsualizar en tiempo real archivos escritos en **Edumark** (`.edu`) y exportarlos directamente como paquetes de **eXeLearning** (`.elpx`).

## Integración con Escola 4.0

Esta extensión es un componente clave utilizado por la extensión [Escola 4.0](https://github.com/gerardfp/escola40) para la producción de materiales didácticos interactivos. 

Mientras que **Escola 4.0** proporciona la plantilla y los bloques de diseño interactivo de alta calidad, **edu2elpx** se encarga de:
- Compilar el contenido escrito en Edumark.
- Generar la vista previa dinámica.
- Empaquetar todo el contenido en el formato nativo `.elpx` listo para abrir en eXeLearning o subir a plataformas educativas.

## Características

- **Vista Previa en Tiempo Real**: Visualiza los cambios de tus archivos `.edu` al instante.
- **Exportación Directa**: Exporta a paquetes `.elpx` con un solo clic o comando.
- **Compilación Integrada**: Lógica de parsing avanzada para interpretar el lenguaje Edumark.
